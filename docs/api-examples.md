# Book Library API

Базовый URL: `http://localhost:8080`

## Эндпоинты

### Авторизация

| Метод | Путь         | Описание              | Защита    |
|-------|-------------|-----------------------|-----------|
| POST  | `/register` | Зарегистрироваться    | —         |
| POST  | `/login`    | Войти, получить JWT   | —         |

### Книги

| Метод   | Путь           | Описание           | Защита    |
|---------|----------------|--------------------|-----------|
| POST    | `/books`       | Создать книгу      | Bearer    |
| GET     | `/books/{id}`  | Получить книгу     | —         |
| GET     | `/books`       | Список всех книг   | —         |
| PUT     | `/books/{id}`  | Обновить книгу     | —         |
| DELETE  | `/books/{id}`  | Удалить книгу      | —         |

> 🔐 `POST /books` требует JWT-токен в заголовке `Authorization: Bearer <token>`. Остальные эндпоинты книг открыты.

---

## Авторизация

### Регистрация

```bash
curl -X POST http://localhost:8080/register \
  -H "Content-Type: application/json" \
  -d '{"username": "alice", "password": "secret123"}'
```

**Тело запроса:**

| Поле     | Тип    | Обязательное | Описание     |
|----------|--------|-------------|--------------|
| username | string | да          | Логин        |
| password | string | да          | Пароль       |

**Ответ `201 Created`:**

```json
{
  "username": "alice"
}
```

**Ошибки:**

| Статус | Когда                           |
|--------|----------------------------------|
| 400    | Невалидный JSON или пустые поля  |
| 409    | Пользователь уже существует      |

---

### Вход (получение JWT)

```bash
curl -X POST http://localhost:8080/login \
  -H "Content-Type: application/json" \
  -d '{"username": "alice", "password": "secret123"}'
```

**Тело запроса:**

| Поле     | Тип    | Обязательное | Описание     |
|----------|--------|-------------|--------------|
| username | string | да          | Логин        |
| password | string | да          | Пароль       |

**Ответ `200 OK`:**

```json
{
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "username": "alice"
}
```

**Ошибки:**

| Статус | Когда                              |
|--------|-------------------------------------|
| 400    | Невалидный JSON или пустые поля     |
| 401    | Неверный логин или пароль           |

> 💡 Токен живёт **24 часа**. Сохрани значение поля `token` — оно понадобится для защищённых эндпоинтов.

## Книги

### Создать книгу

```bash
# Сохрани токен в переменную
TOKEN="eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."

curl -X POST http://localhost:8080/books \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $TOKEN" \
  -d '{"title": "Война и мир", "author": "Лев Толстой", "year": 1869}'
```

**Заголовки:**

| Заголовок      | Значение              | Обязательный |
|----------------|-----------------------|-------------|
| Authorization  | `Bearer <JWT-токен>`  | да          |

**Тело запроса:**

| Поле    | Тип     | Обязательное | Описание          |
|---------|---------|-------------|-------------------|
| title   | string  | да          | Название книги    |
| author  | string  | да          | Автор             |
| year    | int     | да          | Год издания       |

**Ответ `201 Created`:**

```json
{
  "id": 1,
  "title": "Война и мир",
  "author": "Лев Толстой",
  "year": 1869
}
```

**Ошибки:**

| Статус | Когда                              |
|--------|------------------------------------|
| 400    | Невалидный JSON в теле             |
| 401    | Отсутствует или невалидный токен   |

---

### Получить книгу по ID

```bash
curl http://localhost:8080/books/1
```

**Параметры пути:**

| Параметр | Тип   | Описание  |
|----------|-------|-----------|
| id       | int   | ID книги  |

**Ответ `200 OK`:**

```json
{
  "id": 1,
  "title": "Война и мир",
  "author": "Лев Толстой",
  "year": 1869
}
```

**Ошибки:**

| Статус | Когда          |
|--------|----------------|
| 400    | Невалидный id  |
| 404    | Книга не найдена |

---

### Список всех книг

```bash
curl http://localhost:8080/books
```

**Ответ `200 OK`:**

```json
[
  {
    "id": 1,
    "title": "Война и мир",
    "author": "Лев Толстой",
    "year": 1869
  },
  {
    "id": 2,
    "title": "Преступление и наказание",
    "author": "Фёдор Достоевский",
    "year": 1866
  }
]
```

Если книг нет — возвращается пустой массив `[]`.

---

### Обновить книгу

```bash
curl -X PUT http://localhost:8080/books/1 \
  -H "Content-Type: application/json" \
  -d '{"title": "Война и мир", "author": "Лев Толстой", "year": 1873}'
```

**Параметры пути:**

| Параметр | Тип   | Описание  |
|----------|-------|-----------|
| id       | int   | ID книги  |

**Тело запроса:**

| Поле    | Тип     | Обязательное | Описание          |
|---------|---------|-------------|-------------------|
| title   | string  | да          | Новое название    |
| author  | string  | да          | Новый автор       |
| year    | int     | да          | Новый год         |

**Ответ `200 OK`:**

```json
{
  "id": 1,
  "title": "Война и мир",
  "author": "Лев Толстой",
  "year": 1873
}
```

**Ошибки:**

| Статус | Когда                    |
|--------|--------------------------|
| 400    | Невалидный id или JSON   |
| 404    | Книга не найдена         |

---

### Удалить книгу

```bash
curl -X DELETE http://localhost:8080/books/1
```

**Параметры пути:**

| Параметр | Тип   | Описание  |
|----------|-------|-----------|
| id       | int   | ID книги  |

**Ответ:** `204 No Content` (тело пустое)

**Ошибки:**

| Статус | Когда            |
|--------|------------------|
| 400    | Невалидный id    |
| 404    | Книга не найдена |
